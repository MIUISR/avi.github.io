Whatsapp data analysis

In my freetime I wanted to analyse an old Whatsapp group chat and compare it with its new one to see things like difference in activity. Whatsapp offers you to download group chats as .txt-files. I wrote a Python script to read this data in and analyse all messages on things I was interested in. For example I wanted to know who is writing the most messages and words and when everyone is the most active to find the perfect question time.

You can find the complete code on my Github repository!
