Stock data analysis

Since I'm interested in stocks, I wanted to dive deeper into stock analysis. My goal in this private project was to build an stock visualization I can later automate, to send myself daily summaries of stock performances. Therefor I searched for an stock-data API, I can retrieve their values from the current day, week, ... I found the Alpha Vantage API, that was not deprecated and can retrieve realtime stock data like: open, close, low, high. To send the plotted summary via e-mail I used the SMTP protocol to send data to my personal e-mail. For automation reasons, the Python script will wait to plot and send data at 19:00 o'clock, when stock market is closing. To run by itself I built a Docker image to run my script in a container instance. For visualising data I used a so called candlestick graph to present all data in a compact way. In the following you can see a generated plot of the apple stock on 30 July 2019.

You can find the code in my Github repository!
