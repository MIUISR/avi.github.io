Which dog breed are you?

I'm currently working on a project, where a user should be able to upload a picture of his face and receives one of 65 dog breeds which is the most similar to his face shape. Therefore I'm building a CNN to classify dog breeds by an own generated dataset inheriting around 12.000 images. I already designed and developed the user interface created with Django in Python 3, as well as the dataset and an simple CNN, which I'm going to improve next.
